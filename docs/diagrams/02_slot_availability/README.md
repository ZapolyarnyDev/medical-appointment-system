# Блок-схема проверки доступности времени приема

Диаграмма показывает, как система определяет, можно ли записаться к врачу на выбранное время.

Файлы:

- `slot-availability.drawio` - исходник для diagrams.net.
- `slot-availability.png` - изображение для быстрого просмотра.

Что отражено на схеме:

- получение врача и выбранного времени приема;
- проверка существования врача;
- проверка существования времени приема;
- проверка принадлежности времени выбранному врачу;
- проверка статуса и даты;
- итоговое решение: запись разрешена или отклонена.

Диаграмма взята из материалов в директории выше проекта и заменяет временную схему, созданную ранее.
