# Архитектурная диаграмма

Диаграмма показывает слои приложения и основные связи между ними.

Файлы:

- `context.drawio` - исходник для diagrams.net.
- `context.png` - изображение для быстрого просмотра.

Что отражено на диаграмме:

- представление: страницы Thymeleaf и описание программного интерфейса;
- серверная часть: контроллеры, проверка доступа, прикладная логика;
- работа с данными: jOOQ, Flyway и PostgreSQL;
- авторизация: Keycloak, роли и проверка прав.

Диаграмма взята из материалов в директории выше проекта и заменяет временную схему, созданную ранее.
