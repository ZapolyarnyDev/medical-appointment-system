# Диаграмма базы данных

Диаграмма описывает таблицы приложения и связи между ними.

Файлы:

- `er.drawio` - исходник для diagrams.net.
- `er.png` - изображение для быстрого просмотра.

Основные связи:

- врач относится к специализации;
- время приема относится к врачу;
- запись связывает пациента и время приема;
- учетная запись пациента связывает пользователя Keycloak с пациентом;
- учетная запись сотрудника связывает пользователя Keycloak с ролью и, при необходимости, с карточкой врача.

Диаграмма взята из материалов в директории выше проекта и заменяет временную схему, созданную ранее.
