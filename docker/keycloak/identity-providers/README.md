# Шаблоны провайдеров входа

Эта директория содержит заготовки для подключения Keycloak к внешним источникам личности

## Режимы

- `local` — локальные пользователи Keycloak
- `ad` — Active Directory
- `freeipa` — FreeIPA
- `ldap` — общий LDAP-каталог
- `external-oidc` — внешний OIDC-провайдер

## Важное правило

Приложение не подключается напрямую к AD, FreeIPA или LDAP

Пользователь входит через Keycloak, приложение получает OIDC/JWT и работает только с ролями и связями в доменных таблицах
